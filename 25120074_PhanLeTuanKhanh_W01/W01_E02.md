## Exercise 2: Stack by Linked List

### Module 2.1: Basic Stack Implementation (5 points)
Use a Singly Linked List structure to implement a Stack data structure that stores integers (`int`).

Students must define a `StackNode` struct and a `MyStack` struct to hold the state. Write standalone functions that accept the stack state as a parameter:
* `initStack(MyStack& st)`: Initialize the stack (e.g., set the top pointer to null).
* `push(MyStack& st, int x)`: Add element `x` to the top of the stack.
* `pop(MyStack& st)`: Remove the element at the top of the stack (does not need to return a value).
* `top(MyStack st)`: Return the value of the element at the top of the stack. If the stack is empty, return an error value (e.g., `-1`).
* `isEmpty(MyStack st)`: Return `true` if the stack is empty, otherwise return `false`.
* `clearStack(MyStack& st)`: Pop all elements and free allocated memory to prevent memory leaks.

### Module 2.2: "Valid Parentheses" Problem (4 points)
**Description:** Given a string `s` containing just the characters `'('`, `')'`, `'{'`, `'}'`, `'['` and `']'`. Use the `MyStack` structure implemented in Module 2.1 to write a function that determines if the input string is valid.
*Note: Since `MyStack` stores `int`, you can type cast the ASCII code of the character (`char`) to an integer to store it in the Stack, or store the index of that character.*

The string is considered valid if:
1. Open brackets must be closed by the same type of brackets.
2. Open brackets must be closed in the correct order.
3. Every close bracket has a corresponding open bracket of the same type.

**Examples:**
* Input: `s = "()[]{}"` -> Output: `true`
* Input: `s = "([)]"` -> Output: `false`

### Module 2.3: Longest Valid Parentheses (1 point)
**Description:** Given a string `s` containing just the characters `'('` and `')'`. Find the length of the longest valid (well-formed) parentheses **substring** (contiguous).
You must strictly use the implemented `MyStack`, combined with basic loops/conditional statements.

**Examples:**
* Input: `s = "(()"` -> Output: `2` (Explanation: The longest valid substring is `"()"`)
* Input: `s = ")()())"` -> Output: `4` (Explanation: The longest valid substring is `"()()"`)
* Input: `s = ""` -> Output: `0`
