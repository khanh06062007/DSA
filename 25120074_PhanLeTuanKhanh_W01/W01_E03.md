## Exercise 3: Queue by Linked List

### Module 3.1: Basic Queue Implementation (5 points)
Use a Singly Linked List structure to implement a Queue data structure that stores integers (`int`).

Students must define a `QueueNode` struct and a `MyQueue` struct to hold the state. Write standalone functions that accept the queue state as a parameter:
* `initQueue(MyQueue& q)`: Initialize the queue (e.g., set the front and rear pointers to null).
* `enqueue(MyQueue& q, int x)`: Add element `x` to the rear/tail of the queue.
* `dequeue(MyQueue& q)`: Remove the element at the front/head of the queue (does not need to return a value).
* `front(MyQueue q)`: Return the value of the element at the front of the queue. If the queue is empty, return an error value (e.g., `-1`).
* `isEmpty(MyQueue q)`: Return `true` if the queue is empty, otherwise return `false`.
* `clearQueue(MyQueue& q)`: Dequeue all elements and free allocated memory to prevent memory leaks.

### Module 3.2: "Time Required to Buy Tickets" Problem (4 points)
**Description:** There are `n` people standing in line to buy movie tickets, lined up in order from $0$ to $n-1$. You are given a static array `tickets` of size `n`, where `tickets[i]` represents the number of tickets that the `i`-th person wishes to buy.
Each person takes exactly 1 second to buy a ticket. A person can only buy **1 ticket per turn**; after buying, if they still need to purchase more tickets, they must go to the back of the line. If the person no longer needs to buy any tickets, they will leave the line.

Use the `MyQueue` structure written in Module 3.1 to write a function that calculates the time (in seconds) required for the person standing at position `k` (0-indexed) to finish buying all of their tickets.
*(Suggested function signature: `int timeRequiredToBuy(int tickets[], int n, int k)`)*

**Examples:**
* Input: `tickets = {2, 3, 2}`, `n = 3`, `k = 2`
* Output: `6` (Explanation: The queue status per second: `[2,3,2]` -> `[3,2,1]` -> `[2,1,2]` -> `[1,2,1]` -> `[2,1]` -> `[1,1]` -> The 2nd person finishes buying).

### Module 3.3: Catch That Cow (1 point)
**Description:** Farmer Thien is trying to catch an escaped cow. Both are standing on a straight line (1D coordinate axis). The farmer is currently at position $N$, and the cow is standing still at position $K$ ($0 \le N, K \le 100,000$). Every minute, Farmer Thien can perform one of the following two actions:
1. **Walk:** Move 1 step left or right (from position $X$ to $X - 1$ or $X + 1$).
2. **Teleport:** Jump to exactly double his current position (from position $X$ to $2 \times X$).

Using strictly the `MyQueue` structure from Module 3.1 and **dynamic arrays** (using the `new` keyword to store visited states and distances), calculate the minimum number of minutes required for Farmer Thien to reach exactly position $K$. Do not forget to free the allocated memory (using `delete[]`) before the function ends.

**Examples:**
* Input: `N = 5`, `K = 17`
* Output: `4` (Explanation: `5` -> `10` -> `9` -> `18` -> `17`, taking 4 minutes).
